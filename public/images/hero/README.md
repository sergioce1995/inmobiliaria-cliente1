# Imagen Hero de la Página Principal

## Instrucciones

1. **Descarga la foto de Paula Gutiérrez Hurtado**
2. **Renómbrala a**: `paula.jpg`
3. **Colócala en esta carpeta**: `public/images/hero/`

## Formato recomendado

- **Nombre**: paula.jpg
- **Tamaño**: ~900x600 px o mayor
- **Formato**: JPG o PNG
- **Peso**: < 500KB (para carga rápida)

## Fallback

Si la imagen no se encuentra, se mostrará un placeholder de Unsplash automáticamente.
